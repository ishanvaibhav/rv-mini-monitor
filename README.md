# RV-Mini-Monitor

Interactive **RISC-V M-Mode Bare-Metal Monitor Dashboard** built with **React, TypeScript, Vite, and Tailwind CSS**.

This project simulates a low-level RISC-V monitor environment with an interactive terminal, system diagnostics, memory mapping, and hardware-inspired dashboard UI.

---

## 🚀 Features

- 🖥️ Interactive terminal emulator
- ⌨️ Custom shell commands  
  (`help`, `regs`, `pmp`, `top`, `neofetch`, `status`, `date`, etc.)
- ⚙️ Boot sequence simulation
- 🧠 Register dump & monitor diagnostics
- 🗺️ Memory map / payload layout panels
- 🔌 QEMU + VisionFive themed system environment
- 🌙 Clean dark responsive UI
- 📈 Progress / roadmap dashboard
