# RV-Mini-Monitor

Interactive RISC-V M-Mode Bare-Metal Monitor Dashboard built with React, TypeScript, Vite, and Tailwind CSS.

## 🚀 Features

- Interactive terminal emulator
- Custom shell commands (`help`, `regs`, `pmp`, `top`, `neofetch`, `status`)
- Boot sequence simulation
- Memory map dashboard
- QEMU + VisionFive themed interface
- Dark responsive UI

## 💻 Run Locally

```bash
npm install
npm run dev
